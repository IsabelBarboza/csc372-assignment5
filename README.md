# csc372-assignment5 Github Fetch Gallery

This project shows 10 lastest repositories of a Github user.

## Features
- Repository with icon and link
- Description
- Creation date
- Update date
- List of languages
- Number of watchers

## Tools
- HTML
- CSS
- Javascript
- Fetch API
- Font Awesome

## How to use
- Type a Github username in the search box
- Click Search (Gallery show the 10 lastest repos.)

## Code guidelines
- All CSS in in style.css
- All JS is in script.js
- Event handling in JS 
- Comments on the code

## Deployment
The site is deployed using Github Pages
